# fe-search-meli

## Needed to run the project

- [NodeJS](https://nodejs.org/es/download/) v. 14.0.0 or lower.
- [Yarn](https://classic.yarnpkg.com/en/docs/install/) 1.22.4 or lower.
- Nodemon globally. (`yarn global add nodemon`)

## Setup

1. `cd client`
2. `yarn install`
3. `cd ..`
4. `yarn install`
5. `yarn dev`

It's running!
